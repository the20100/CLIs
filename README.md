# CLI Index

All CLIs are Go projects targeting the `the20100` GitHub account. They output JSON when piped and human-readable tables in a terminal.

---

## Google APIs

| CLI | Repo | Description |
|-----|------|-------------|
| g-docs-cli | [gdocs-cli](https://github.com/the20100/gdocs-cli) | Google Docs — create documents, read content, insert text, find & replace, delete ranges |
| g-drive-cli | [gdrive-cli](https://github.com/the20100/gdrive-cli) | Google Drive v3 — list, upload, download, and manage files and folders |
| g-indexing-cli | [g-indexing-cli](https://github.com/the20100/g-indexing-cli) | Google Indexing API — notify Google Search when pages are created, updated, or deleted |
| g-search-console-cli | [gsc-cli](https://github.com/the20100/gsc-cli) | Google Search Console — query search performance, sitemaps, and URL inspection |
| g-sheets-cli | [gsheets-cli](https://github.com/the20100/gsheets-cli) | Google Sheets — read/write spreadsheet data, manage sheets and tabs |
| g-slides-cli | [gslides-cli](https://github.com/the20100/gslides-cli) | Google Slides — create and modify presentations programmatically |
